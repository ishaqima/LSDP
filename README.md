## 1. 快速创建github远程仓库
> 在github网站上，Repositories->New，新建项目；
> 在本地文件夹中，git clone ...，将远程仓库克隆到本地；
> 克隆之后直接在本地文件夹中修改、更新即可；
> [[1]github入门教程](https://www.jianshu.com/p/7f8c80056233)

## 2. Markdown常用语法
> 标题：一级(#)、二级(##)、...、六级（######）；
> 访问链接：[文字](网址)；
> 引用：一级(>)、二级(>>)，可嵌套引用；
> 无序列表：*、+、-；
> 有序列表：1.、2.、...；
> 列表状态：
- [ ] 不勾选
- [x] 勾选
