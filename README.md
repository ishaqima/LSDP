# 常用知识快速查找
## 1. 快速创建github远程仓库
> 在github网站上，Repositories->New，新建项目；  
> 在本地文件夹中，git clone ...，将远程仓库克隆到本地； 
> 克隆之后直接在本地文件夹中修改、更新即可；  
> [参考博客](https://www.jianshu.com/p/7f8c80056233)

## 2. Markdown常用语法  
> 换行：在一行结尾添加两个空格;  
> 分割线：连续三个***、---、___  
***
> 标题：一级(#)、二级(##)、...、六级（######）；  
> 访问链接：```[文字](网址)```；  
> 引用：一级(>)、二级(>>)，可嵌套引用；  
> 无序列表：*、+、-；  
> 有序列表：1.、2.、...；  
> 列表状态：
```- [ ] 不勾选、- [x] 勾选```  
- [ ] 不勾选  
- [x] 勾选  
> 内联图片：
```![Alt text](./picture/xyjy.jpg)```  
![Alt text](./picture/xyjy.jpg)  

> 代码：三个顿号括起来的区域  
```
%% matlab代码
% 清除环境变量
clear all
clc

% 显示图片
I = imread('happy.jpg');
imshow(I);
```
